📝 Measurement Procedure

1. Setup & Validation

Verify Ambient Noise (Receiver scan with Antenna disconnected / terminated)
Connect Antenna (e.g., Teseq UPA 6108) to Receiver RF Input
Check Setup Photos for cable routing (minimize loops)

2. Receiver Config (ESCI 7)

Set Frequency Range (Start/Stop) or Span (30-1000 MHz)
Set RBW = 120 kHz (Standard for CISPR 30M-1G)
Select Detector: Peak (fast scan) or Quasi-Peak (final)

3. Measurement & Maximization

Scan Height: Move antenna 1m to 4m (Max Hold)
Rotate Turntable: 0 to 360 degrees (find max azimuth)
Check Polarization: Test both Horizontal & Vertical
Final: Perform QP Meas on worst-case frequencies
